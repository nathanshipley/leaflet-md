# Second Window Demo

Open this file alongside the first sample to check that multiple document windows can stay open at the same time.

## Things to Try

- Open this file with `File > Open...`
- Open the first sample too
- Drag either file onto the app icon in the Dock
- Drag a markdown file onto an existing window

## Expected Feel

Each file should open in its own normal macOS document window, more like TextEdit than a single-window utility.
